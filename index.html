// how-it-works

Three stages. Zero manual steps.

This site is intentionally plain — the interesting part is the pipeline shipping it, not the HTML.

Get Code

Jenkins runs checkout scm to pull the latest commit straight from Git.

Build Image

The Dockerfile packages these files into a small nginx image, tagged and ready to run.

Deploy

Jenkins stops the old container and starts a new one from that image, live on port 80.
